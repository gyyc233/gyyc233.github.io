LINUX下Cpp开发基本语句

发表于 2021-06-01 更新于 2021-08-29 阅读次数：

基本语句

使用环境为ubantu 20.04

linux系统介绍

重点关注

指令和选项

重要指令讲解

pwd -print current workingdirectory

ls - list directory contents

  • 用法2： ls [路径]

含义：列出指定路径下的所有文件/文件夹的名称

  1. 绝对路径：相对于root目录的路径
  2. 相对路径：相对与当前目录的路径

  • 用法3： ls 选项

    含义：列出指定路径下的文件/文件夹的名称，并以指定格式进行显示

在输入 ls -a前后，发现多了一些.开头的文件。

  • ctrl+l : 在终端输入快捷键 ctrl+l 使终端清屏。

cd -change directory

  • 作用：切换当前工作目录
  • 用法1：cd;cd ~
  • 用法2： cd [相对路径]
  • 用法3：cd [绝对路径]

mkdir -make direcories

  • 作用：创建目录
  • 用法1：mkdir 路径
  • 用法2：mkdir -p 路径
  • 含义：一次性创建多层不存在的目录
  • 用法3：mkdir 路径1 路径2
  • 含义：一次创建多个目录

tree的使用

ubantu 默认是没有tree指令来显示文件树的，需要安装

终端输入 sudo apt-get install tree 进行安装

常用语句：

touch -change file timestamps

作用：创建新文件

  • 用法1：touch 路径
  • 用法2：touch 文件1 文件2

rm -remove files or directories

作用：删除文件/目录

  • 用法1：rm [选项]
